MINECRAFT COMMANDS

OP Commands

Game Mode

LEGENDARY

Switch between Creative and Survival modes instantly. Perfect for building without limitations!

/gamemode creative @s

Give Item

EPIC

Summon a fully enchanted diamond pickaxe with Unbreaking III and Efficiency V.

/give @s diamond_pickaxe{Enchantments:[{id:"minecraft:unbreaking",lvl:3},{id:"minecraft:efficiency",lvl:5}]}

Set Spawn

RARE

Set the world spawn point to your current location. Players will respawn here after death.

/setworldspawn ~ ~ ~

Teleport

LEGENDARY

Instantly teleport to any coordinates. Great for fast travel across your world!

/tp @s 100 64 200

Clear Inventory

UNCOMMON

Instantly clear all items from your inventory. Useful for cleaning up quickly!

/clear @s

Time Set

RARE

Change the world time instantly. Set it to day, night, or any time value (0-24000).

/time set day
